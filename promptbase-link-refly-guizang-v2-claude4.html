快速开始指南

三步即可开始使用Promptbase，快速提升您的AI应用效果

步骤 1: 安装配置

# 使用npm安装
npm install @microsoft/promptbase
# 或使用pip安装
pip install promptbase

步骤 2: 初始化项目

# 创建新项目
promptbase init my-prompts
# 配置API密钥
promptbase config set api-key YOUR_API_KEY

步骤 3: 创建提示词

# 创建新的提示词
promptbase create customer-service
# 部署到生产环境
promptbase deploy --env production

示例提示词模板

// customer-service.prompt
---
name: 客户服务助手
version: 1.0.0
tags: [customer-service, support]
---
你是一位专业的客户服务代表。
请以友好、耐心的态度回答客户问题。
客户问题: {{customer_question}}
产品信息: {{product_info}}
